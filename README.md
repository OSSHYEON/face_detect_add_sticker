# face_detect_sticker_app
<br>

미완성! 아직 작성 중이에요 🛠
- dlib 라이브러리 사용해 얼굴 검출 하는 프로그램이에요
- 영상 읽어와 큐에 담는 스레드와, 큐에서 꺼내 연산하는 스레드가 있어요
- 원하는 스티커 버튼을 누르면 이미지를 바꿔 담아요
- 캡쳐 버튼을 누르면 스티커를 씌운 이미지를 보관할 수 있어요! ( 버튼을 누르고 사진을 저장하기까지 딜레이가 있어요🥲 )
